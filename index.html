HappyLifts

This Full Stack MERN application enables users to search and customize workout plans targeting specific muscles. It features robust front and back-end validations for enhanced data integrity and user experience. The front end utilizes React.js with hooks for state management, React Router for navigation, and TanStack React Query for data handling. The design is fully responsive, ensuring a seamless user experience on all devices.

On the back end, Node.js and Express.js manage user authentication via middleware for secure access to protected routes. MongoDB, coupled with Mongoose ORM, ensures reliable data storage and validation.

Built with: JavaScript, React.js (using useState, useEffect, custom hooks, react-router-dom, and tanstack/react-query), Node.js, Express.js (with middleware for route protection), Mongoose (with validations), MongoDB, CSS Modules
